LazerTank

A brain-teaser capture the flag puzzle game

Available Commands:

arrow keys - move

move in the direction of the arrow key

space bar - shoot

shoot the tank's laser once

numeric keys - shoot multiple times

shoot the tank's laser the specified number of times

mouse left click - move to

Moves the tank to the selected square if possible

mouse drag - move

Define a path for the tank by moving the mouse while holding down the left mouse button

c - Capture the flag

Type a c to move the tank to the flag if a possible path exists

Ctrl - focus

Switch focus from the last pending move to the tank. Entering commands while focused on the tank replaces any pending moves

backspace - Erase

Erase the last pending move. Ctrl-backspace erases all pending moves

Esc - Menu

Open the menu

mouse right click - Menu with context

Open the menu with additional context for the selected square
