LazerTank

A brain-teaser capture the flag puzzle game.

Available Commands

arrow keys

move

space bar

shoot once

mouse left click

Move to the selected square. I.e. Auto-generate a path to the mouse position if an open path can be found.

c

capture the flag. I.e. Auto-generate a path to the flag if an open path can be found.

numeric keys

shoot a specific number of times

backspace

erase the last pending move

Ctrl

switch focus from the last pending move to the tank. Entering commands while focused on the tank replaces any pending moves.

Esc

Open the menu

mouse right click

Open the menu with mouse position context
